## Features

- __Async:__ Use Async Rust
- __Stream:__ Support Streaming
- __Diff:__ Diff support
- __CRD:__ Support CRD
- __Generic:__  Use Rust Generics

Following OS are supported:
* Linux
* MacOs

Windows support will be coming in future
